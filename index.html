How to use the application

Schemas: create or edit note schemas on the Schemas screen. Each schema defines a set of fields (text, number, date, boolean).

Notes: select a schema on the Notes screen, then create and edit notes that follow the selected schema. Use the search box to filter notes.

Settings: use Export / Import to backup or restore data as JSON. The "Clear all data" button removes all schemas and notes from localStorage.
