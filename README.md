# Structured Notes Editor
Lab 2 – GUI (JavaScript, HTML, CSS)

## Overview
The project implements a simple structured notes editor.  
Users can define note schemas (templates), create notes based on these schemas, search notes, and export/import data as JSON.

The project follows the general rules for Lab 2:
- Modular code (separate JS modules).
- GUI separated from business logic.
- At least 4 screens.
- At least 20 GUI controls.
- At least 10 event handlers.
- Basic unit tests.
